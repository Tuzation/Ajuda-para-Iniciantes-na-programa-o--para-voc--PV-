Qual é o Melhor Aplicativo para Começar

Para começar na área de programação, primeiro precisará de um bom aplicativo te ajudar no começo.

Eu indico o Visual Studio Code clique no link a seguir para baixar o VScode para qualquer sistema operacional.

Recomendo ele porque ele facilita sua vida com funções que vão te ajudar no início ele também tem extenções que voçê pode usa

Configuração do Aplicativo

Depois de baixar o aplicativo, entre nele e clique no símbolo de configurção depois em settings ou Ctrl+ e pesquise 3 parâmetros, word wrap, color theme, font size e o auta save. Entre nas extenções pesquise seu idioma nas extenções e instale ele para que a extenção traduza a linguagem que está VScode que normalmente é inglês.

  • Word Wrap = Colocar no "ON" para fazer com que ele quebre linha.
  • Color Theme = Mudar a cor de fundo do jeito que tu querer.
  • Font size = Tamanho da letra coloca 14 ou 20.
  • Auto Save = Ative o auto save para que você não fique precisando usar "Ctrl s" toda hora para salvar seu código.
